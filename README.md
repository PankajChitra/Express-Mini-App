# Express Mini App Monorepo

This repository contains both backend and frontend apps.

## Structure

- Backend/: Express API server
- Frontend/: Vite + React client

## Quick Start

### 1) Backend

```bash
cd Backend
npm install
npm run dev
```

### 2) Frontend

```bash
cd Frontend
npm install
npm run dev
```

## Notes

- Backend environment files are expected inside Backend/ (for example: Backend/.env).
- Frontend API base URL is configured in Frontend/src/api/axios.js.
